# Biblioteca Virtual UniFECAF

Este projeto foi desenvolvido como atividade acadêmica da disciplina de Desenvolvimento Web, com o objetivo de criar a interface de um site para a biblioteca virtual da UniFECAF, seguindo o protótipo de baixa fidelidade disponibilizado pelo professor.

## Objetivo do projeto

Desenvolver um site institucional para a Biblioteca Virtual UniFECAF, permitindo a apresentação visual de livros, unidades da instituição e informações de contato, utilizando HTML e CSS.

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

O projeto foi organizado com os seguintes arquivos e pastas:

- `index.html` → estrutura principal do site
- `style.css` → estilização visual da página
- `img/` → pasta contendo imagens utilizadas no projeto

## Funcionalidades desenvolvidas

- Barra de navegação
- Banner principal
- Seção “Sobre a Nossa Biblioteca”
- Banner intermediário
- Seção “Nossos Livros”
- Seção “Nossas Unidades”
- Seção “Contatos”

## Observações

O projeto foi desenvolvido com foco em organização de código, identidade visual institucional e aproximação com o layout proposto no protótipo.
